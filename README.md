# Acme Web Design – Multi-Page Responsive Website

This project is a responsive multi-page business website built using **HTML5** and **CSS3**.
It represents a fictional web design company and includes a homepage, about page, and services page.

The layout is built using floats and responsive media queries to adapt across different screen sizes.

---

## Pages Included

* Home page (`index.html`)
* About page (`about.html`)
* Services page (`services.html`)

---

## Features

* Responsive navigation header
* Hero showcase section on homepage
* Newsletter subscription section
* Services list with pricing
* Sidebar with quote form
* Multi-column layout
* Mobile responsive design
* Clean business-style UI

---

## Technologies Used

* HTML5
* CSS3
* Media Queries for responsiveness
* Float-based layout system

## How to Run the Project

1. Download or clone the project folder
2. Keep all images in the same directory as the HTML files
3. Open `index.html` in any browser

No server or installation required.

---

## Responsive Behavior

The site becomes mobile-friendly when screen width is **768px or less**:

* Navigation stacks vertically
* Boxes become full-width
* Sidebar moves below main content
* Forms expand to full width
